# Suggestion: Breadcrumb Navigation

**Priority:** Low  
**Status:** Pending  
**Created:** 2026-04-16

---

## Description

Add breadcrumb navigation to the docs viewer content area to help users understand their current location within deeply nested spec trees.

## Proposed Design

```
spec / 02-coding-guidelines / 01-cross-language / 02-boolean-principles / 01-is-has-prefix.md
```

- Each segment is clickable (navigates to that folder/file)
- Truncate long paths with ellipsis on mobile
- Use semantic tokens for styling (muted text, accent on hover)
