**Priority:** Medium  
**Status:** Pending

## Description

Perform full end-to-end browser testing of the docs viewer across all 4 view modes (Preview, Source, Edit, Split) plus the download folder as ZIP feature.

## Test Cases

1. Navigate spec tree — expand/collapse folders, select files
2. Preview mode — verify markdown renders correctly (tables, code blocks, checklists)
3. Source mode — verify raw markdown with syntax highlighting
4. Edit mode — verify editing capability
5. Split mode — verify side-by-side rendering
6. Download — verify ZIP generation and download
7. Theme toggle — verify light/dark mode switching
8. Keyboard navigation — verify arrow keys, Enter, Ctrl+K, Escape
